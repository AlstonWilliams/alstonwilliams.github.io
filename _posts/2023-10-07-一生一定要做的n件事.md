### 💰 财务与事业 (Finance & Career)
- [x] 升职到预期的职位
- [x] 拿到预期的薪水
- [x] 找到喜欢的工作，要求是薪水不能过低，并且工作不过于饱和，能有自己的时间去思考一些事情
- [ ] 能够有稳定的投资组合，一年收益 8%-10%
- [ ] 有一门自己的生意
- [ ] 形成体系化的技术体系，在未来有立足之地

### 🏠 家庭、情感与传承 (Family, Emotion & Legacy)
- [x] 有一个喜欢的家
- [x] 记录自己和爱人的生活，并整理成一个相册或者回忆录
- [x] 给摩卡可可最好的生活。在他们去世之后把他们埋葬在离我们近的地方
- [ ] 跟爱的人厮守一生
- [ ] 建立家庭智库，记录在投资、理财、经济、文化、人文等方面的思考
- [ ] 每年生日或者年底记录一段视频，聊聊方面的变化，未来对宝宝的期待

### 🌱 个人成长与修行 (Self-Growth & Mastery)
- [x] 早睡早起，运动，让自己的精力跟得上，不至于出去玩的时候很累扫兴
- [x] 形成有品位的个人穿搭以及个人气质
- [x] 跟随自己的内心过一生，并有不惧异常眼光的镇定
- [x] 从书籍、短视频、电视剧中感受这个世界的美
- [ ] 学习一门外语并能流畅沟通
- [ ] 构建“一生之书” 100 本精选库
- [ ] 每 3-5 年尝试一次为期一周的彻底静默，不带电子设备，只带纸笔进入山间或极简空间
- [ ] 利用体系化的技术能力，写一段程序将一生的理性数据（如投资收益、运动心率、自驾经纬度）转化为一幅不断演化的抽象数字艺术

### 🌍 旅行、探索与深度连接 (Exploration & Connection)
- [x] 能够自驾游
- [ ] 游览 30+ 国家。记录当地的醒来时刻。带回有纪念意义的实物 (进度: 3/30)
- [ ] 实现跨国深度沟通，找当地资深人士 1v1 沟通，交换对方对当地或者行业的见解
- [ ] 尝试跳伞、潜水
- [ ] 体验一次真正的美式圣诞节

### 🤝 社会价值与利他 (Social Impact)
- [x] 做一些爱心慈善活动，比如支援大山中的孩子，或者尝试联合国的一些公益组织
- [ ] 帮助一个人实现他的梦想，或者帮助职场新人
